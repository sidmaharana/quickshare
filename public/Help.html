How to Use

How to Send Files

Click on "Upload File" and select the file from your device.

A 6-digit code will be generated after uploading.

Share this code with the recipient for them to access the file.

How to Receive Files

Enter the 6-digit code in the "Download" section.

Click "Retrieve File" to start the download.

Supported File Types & Size

All file types are supported (documents, images, videos, compressed files, etc.).

Maximum file size: Up to 500MB (subject to change).

Troubleshooting

Issue: Code not working?

Ensure the sender has shared the correct code.

Check if the file has expired or been deleted.

Issue: Unable to upload files?

Check your internet connection.

Ensure the file size is within the allowed limit.
